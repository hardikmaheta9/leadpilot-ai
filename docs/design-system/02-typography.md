# LeadPilot Design System – Typography

## Purpose

Typography defines the visual hierarchy of LeadPilot AI.

Every screen, dashboard, table, form, report, and AI workspace must follow these typography guidelines to maintain a consistent user experience.

---

# Primary Font

**Font Family**

Inter

Fallback:

Segoe UI, Roboto, Helvetica, Arial, sans-serif

Example:

```css
font-family: "Inter", "Segoe UI", Roboto, Helvetica, Arial, sans-serif;
```

# Rules

✅ Always use Inter.

✅ Never mix multiple font families.

✅ Never use more than five font sizes on one screen.

✅ Use bold only for emphasis.

✅ Use muted color for secondary information.

---

# Future Enhancements

The typography system should support:

- Light Theme
- Dark Theme
- Accessibility (WCAG AA)
- Responsive Font Scaling
- High-DPI Displays
