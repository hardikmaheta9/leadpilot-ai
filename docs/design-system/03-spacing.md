# LeadPilot Design System – Spacing

## Purpose

Spacing creates consistency and improves readability across the application.

---

# Base Unit

8px

LeadPilot follows an 8-point spacing system.

---

# Scale

| Token | Size |
|--------|------|
| xs | 4px |
| sm | 8px |
| md | 16px |
| lg | 24px |
| xl | 32px |
| 2xl | 48px |
| 3xl | 64px |

---

# Card Padding

24px

---

# Form Spacing

Label → Input

8px

Input → Input

20px

Section → Section

32px

---

# Table

Header Padding

16px

Row Padding

14px

---

# Dashboard

Card Gap

24px

Widget Gap

32px

---

# Border Radius

Buttons

12px

Cards

16px

Inputs

12px

Badges

999px

---

# Rules

Never use arbitrary spacing.

Use only the spacing scale.

Always align components to the grid.
